java 运行版本问题

在一个Web项目中，点击一个功能，出现404错误，查看tomcat的日志，发现下面错误Unsupported major.minor version 52.0，该问题的原因就是编译的java版本是52.0，即java 1.8，而运行的版本不支持这个版本。

解决的办法，在编译的时候指定java版本，
javac -target 1.6 HelloWorld.java

在eclipse中，设置java的compile版本即可。
Window -> Preferences -> Java -> compiler -> compiler compliance level.
